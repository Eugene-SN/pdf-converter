-----|---------------------------------------------------------------------------|
| 1    | User ID (范围：01h 至 05h)                                                |

#### Parameter Definitions

| Parameter Value | Description                                                                 |
|-----------------|-----------------------------------------------------------------------------|
| 01h             | 使能状态：00h - 禁用，01h - 启用                                         |

| 06h             | 用户名（长度：17 字节）                                                |
| 07h             | 用户加密：00h - 加密，01h - 非加密                                     |
